# Key Equations - Lecture 2

## Wave Optics
$$c = f\lambda = \frac{\omega}{k}$$

Constructive interference (Young's double-slit):
$$S_1P - S_2P = n\lambda$$

Destructive interference (Young's double-slit):
$$S_1P - S_2P = \left(n + \frac{1}{2}\right)\lambda$$

Light intensity (wave interpretation):
$$I = \frac{1}{2}c\epsilon_0 E_0^2$$

Bragg's law (diffraction):
$$2d\sin\theta = n\lambda$$

## Photons
Photon energy:
$$E_{photon} = hf = \frac{hc}{\lambda}$$

Photon momentum:
$$p_{photon} = \frac{h}{\lambda} = \frac{E_{photon}}{c}$$

Photoelectric effect:
$$KE_{max} = hf - \Phi = hf - hf_0 = eV_0$$

Work function threshold:
$$\Phi = hf_0$$

Light intensity (photon interpretation):
$$I = \Gamma_{ph} hf$$

Photon flux density:
$$\Gamma_{ph} = \frac{\Delta N_{ph}}{A \Delta t}$$

## De Broglie Relations
$$\lambda = \frac{h}{p} \quad \text{or} \quad p = \frac{h}{\lambda}$$

Electron kinetic energy from voltage:
$$KE = \frac{p^2}{2m_e} = eV$$

Electron wavelength from accelerating voltage:
$$\lambda = \frac{h}{\sqrt{2m_e eV}}$$

## Physical Constants
- $h = 6.63 \times 10^{-34}$ J·s (Planck's constant)
- $\hbar = h/2\pi = 1.05 \times 10^{-34}$ J·s
- $c = 3 \times 10^8$ m/s (speed of light)
- $e = 1.60 \times 10^{-19}$ C (electron charge)
- $m_e = 9.11 \times 10^{-31}$ kg (electron mass)
- $\epsilon_0 = 8.85 \times 10^{-12}$ F/m (vacuum permittivity)
