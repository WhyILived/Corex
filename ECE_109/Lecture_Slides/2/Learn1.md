# Learn1: Wave Interference and Young's Double-Slit Experiment
> Estimated time: ~30 minutes

## Section 1: Light as an Electromagnetic Wave
- Light is an electromagnetic (EM) wave with electric field $E_y$ and magnetic field $B_z$ perpendicular to each other and to direction of propagation
- The simplest non-trivial light description is a monochromatic plane wave
- The general wave equation in 1D: $\frac{\partial^2}{\partial z^2} E(z,t) - \frac{1}{c^2}\frac{\partial^2}{\partial t^2}E(z,t) = 0$
- The general form of a traveling wave: $E(\vec{r}, t) = E_0 \cos(\vec{k} \cdot \vec{r} - \omega t + \phi_0)$
- $\vec{k} = \hat{x}k_x + \hat{y}k_y + \hat{z}k_z$ is the wave vector with $|\vec{k}| = \frac{2\pi}{\lambda} = \frac{\omega}{c}$
- $c = f\lambda = \frac{\omega}{k}$ is the speed of light (phase velocity)

## Section 2: Young's Double-Slit Experiment
- Light passing through two slits (S1 and S2) creates an interference pattern on a screen
- Constructive interference (bright fringes): path difference $S_1P - S_2P = n\lambda$ where $n = 0, 1, 2, ...$
- Destructive interference (dark fringes): path difference $S_1P - S_2P = (n + \frac{1}{2})\lambda$
- The intensity pattern follows $I = \frac{1}{2}c\epsilon_0|\vec{E}_1 + \vec{E}_2|^2$
- This demonstrates that light is a wave because interference only occurs when waves are combined

## Key Equations
$$c = f\lambda = \frac{\omega}{k}$$

Where:
- $c$ = speed of light ($3 \times 10^8$ m/s)
- $f$ = frequency (Hz)
- $\lambda$ = wavelength (m)
- $\omega$ = angular frequency ($2\pi f$)
- $k$ = wavenumber ($\frac{2\pi}{\lambda}$)

$$S_1P - S_2P = n\lambda \quad \text{(constructive interference)}$$

Where:
- $S_1P, S_2P$ = path lengths from slits to point P
- $n$ = integer (order of interference maximum)

$$S_1P - S_2P = \left(n + \frac{1}{2}\right)\lambda \quad \text{(destructive interference)}$$

$$I = \frac{1}{2}c\epsilon_0|\vec{E}_1 + \vec{E}_2|^2$$

Where:
- $I$ = light intensity (W/m²)
- $\epsilon_0$ = vacuum permittivity ($8.85 \times 10^{-12}$ F/m)
- $\vec{E}_1, \vec{E}_2$ = electric field vectors from each slit
