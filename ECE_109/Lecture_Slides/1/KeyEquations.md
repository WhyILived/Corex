# KeyEquations: Lecture 1

## Wave Equation
$$\frac{\partial^2}{\partial z^2}E(z,t) - \frac{1}{c^2}\frac{\partial^2}{\partial t^2}E(z,t) = 0$$

## Wave Properties
$$\omega = 2\pi f$$
$$k = \frac{2\pi}{\lambda}$$
$$v = \frac{\omega}{k} = f\lambda$$

## Energy Quantization
$$E = hf = \frac{hc}{\lambda} = \hbar\omega$$

## Photoelectric Effect
$$K_{E,max} = hf - \Phi = eV_0$$
$$f_0 = \frac{\Phi}{h}$$

## Photon Momentum
$$p = \frac{h}{\lambda} = \frac{E_{ph}}{c}$$

## Light Intensity and Photon Flux
$$I = \frac{1}{2}c\epsilon_0 E_0^2$$
$$I = \Gamma_{ph}hf$$
$$\Gamma_{ph} = \frac{\Delta N_{ph}}{A \Delta t}$$

## Variable Definitions
- $E$ = photon energy (J)
- $h = 6.6 \times 10^{-34}$ J·s (Planck's constant)
- $\hbar = h/2\pi = 1.05 \times 10^{-34}$ J·s (reduced Planck's constant)
- $f$ = frequency (Hz)
- $\lambda$ = wavelength (m)
- $\omega$ = angular frequency (rad/s)
- $c = 3 \times 10^8$ m/s (speed of light)
- $K_{E,max}$ = maximum electron kinetic energy (J)
- $\Phi$ = work function (J or eV)
- $e = 1.6 \times 10^{-19}$ C (electron charge)
- $V_0$ = stopping voltage (V)
- $f_0$ = threshold frequency (Hz)
- $p$ = photon momentum (kg·m/s)
- $I$ = light intensity (W/m²)
- $\Gamma_{ph}$ = photon flux density (photons/m²/s)
- $\epsilon_0 = 8.85 \times 10^{-12}$ F/m (permittivity of free space)
