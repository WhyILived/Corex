# Learn1: Crystal Structure and Bravais Lattices
> Estimated time: ~30 minutes

## Section 1: Crystal Lattice Concepts
- A crystalline solid is a periodic collection of atoms with long-range order
- A **lattice** is an infinite periodic array of geometric points in space, without any atoms
- A **basis** is an identical group of atoms (or molecules) placed at each lattice point
- A **unit cell** is the most convenient small cell that carries the properties of the crystal; repetition in 3D generates the whole crystal

## Section 2: Crystal Systems and Bravais Lattices
There are 14 kinds of unit cells that can fill 3D space, organized into 7 crystal systems:

### Cubic System (a = b = c, α = β = γ = 90°)
- **Simple cubic**: 1 atom/cell, coordination 6, packing factor 0.52 (no common metals except Po)
- **Body-centered cubic (BCC)**: 2 atoms/cell, coordination 8, packing factor 0.68 (α-Fe, Cr, Mo, W)
- **Face-centered cubic (FCC)**: 4 atoms/cell, coordination 12, packing factor 0.74 (Al, Cu, Ag, Au, Pb)

### Tetragonal System (a = b ≠ c, α = β = γ = 90°)
- **Simple tetragonal**
- **Body-centered tetragonal**

### Orthorhombic System (a ≠ b ≠ c, α = β = γ = 90°)
- **Simple orthorhombic**
- **Body-centered orthorhombic**
- **Base-centered orthorhombic**
- **Face-centered orthorhombic**

### Rhombohedral System (a = b = c, α = β = γ ≠ 90°)
- Arsenic, boron, bismuth, antimony, mercury (< –39°C)

### Hexagonal System (a = b ≠ c, α = β = 90°, γ = 120°)
- Cadmium, magnesium, zinc, graphite

### Monoclinic System (a ≠ b ≠ c, α = β = 90°, γ ≠ 90°)
- α-Selenium, phosphorus, lithium sulfate, tin fluoride

### Triclinic System (a ≠ b ≠ c, α ≠ β ≠ γ ≠ 90°)
- Potassium dicromate

## Section 3: Diamond and Zinc Blende Structures
- **Diamond structure**: Each Si atom bonds tetrahedrally with 4 neighbors (covalent bond directional nature)
  - 8 atoms per unit cell (FCC lattice with basis of 2 atoms)
  - Examples: C (diamond), Si, Ge, α-Sn
- **Zinc blende structure**: Like diamond but with alternating Ga and As atoms at lattice points
  - Examples: GaAs, ZnS, many III-V semiconductors
  - Also called zinc sulfide structure

## Key Equations

**Lattice parameter relationships:**
$$a = 4R/\sqrt{2} = 2R\sqrt{2} \text{ (for FCC)}$$

Where:
- $a$ = lattice parameter (nm)
- $R$ = atomic radius (nm)

**Atomic packing factor (APF):**
$$APF = \frac{\text{Volume of atoms in unit cell}}{\text{Volume of unit cell}}$$

For FCC: $APF = 0.74$ (74% of volume is atoms)

**Atomic concentration:**
$$n_{at} = \frac{x}{a^3}$$

Where:
- $x$ = number of atoms per unit cell
- $a$ = lattice parameter
